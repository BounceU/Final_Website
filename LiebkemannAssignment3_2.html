  • Hello! I'm Ben and I'm starting a blog! I've got big plans for this. For this first post I'm just going to explain a little bit about how this works.

    1. There are no comments, sorry. I'll add those later but I just want to be able to start making posts right now.
    2. You might notice that this post is hardcoded into the webpage. I'm not going to keep editing the html and re-uploading this webpage to github every time I want to make a post, so I'm going to make it so that I just have a separate file that houses all of my posts. This website will have an html script that will check that file and show you the posts inside of it.
    3. You can click the image on the right to enlarge/(ensmall?) it.
    4. There will be more posts coming so be on the look out!

    If you want a little bit more detail on how the blog file system is going to work I'll probably make a program to write a post and put it in a file with the following format:

    "<image url, preferrably one in the images folder>"<Text of the post with markups>

    It's a pretty simple system, I just don't want to get into JSON or XML files because they'll just end up being bigger files for no reason since no one is ever going to be actually going into the files. The only place they'll be shown will be here, and they'll be parsed here.
